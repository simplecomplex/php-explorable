# (PHP) Explorable
<small>composer namespace: simplecomplex/**explorable**</small>

## Foreach'ing protected members made simple

An interface, base class and traits making it a breeze to expose a class'
protected members.

Extending `Explorable` facilitates:
- `isset()`
- `count()`
- `foreach (...`
- `var_dump()`
- `toObject()`, `toArray()`
- `json_encode()`
